# 参考文献格式检查 Skill

检查参考文献格式是否符合国科大硕士论文要求（基于 GB/T 7714—2015），并验证英文标点符号后是否有空格。

## 使用方式

用户可以通过以下两种方式提供参考文献：

1. **粘贴文本**：直接在对话中粘贴参考文献列表。
2. **指定文件**：提供文件路径（支持 `.md`、`.txt`、`.tex`、`.docx`）。

收到参考文献后，逐条检查，最后汇总所有问题。

---

## 一、通用格式规则

GB/T 7714—2015 标准著录格式：

```
主要责任者. 题名: 其他题名信息[文献类型标识/文献载体标识]. 其他责任者. 版本项. 出版地: 出版者, 出版年: 引文页码[引用日期]. 获取和访问路径. 数字对象唯一标识符.
```

### 通用规则

1. 每个著录项之间以 `. `（英文句号 + 空格）分隔（中文文献同样使用半角标点）。
2. 中文文献标点符号统一使用**半角**：`.` `,` `:` `;` 等（与样例保持一致）。
3. 责任者（作者）之间用 `, `（逗号 + 空格）分隔。
4. 题名后紧跟文献类型标识，如 `[M]`、`[J]`。
5. 出版地、出版者、出版年为必填项。
6. 页码前用 `: `（冒号 + 空格）。
7. 每个参考文献条目以 `.` 结尾。

---

## 二、英文标点符号空格规则（重要）

**所有英文文本中，英文标点符号后面必须有且仅有一个空格，唯一例外是条目最后的 `.`。**

适用范围：参考文献中出现的所有英文文本，包括但不限于：
- 英文作者名
- 英文题名/书名
- 英文期刊名
- 英文出版地/出版者

具体要求：

| 标点 | 规则 | 正确示例 | 错误示例 |
|------|------|----------|----------|
| `.` | 后必须有空格（除最后） | `Anat Embryol(Berl), 1990, 181: 67-73.`（最后的.不要求后续空格） | `Anat Embryol(Berl),1990,181:67-73.` |
| `,` | 后必须有空格 | `Bravo H, Olavarria J, Torrealba F.` | `Bravo H,Olavarria J,Torrealba F.` |
| `:` | 后必须有空格 | `Geneva: WHO, 1970.` | `Geneva:WHO, 1970.` |
| `;` | 后必须有空格 | (如有分号使用场景) | — |
| `?` `!` | 后必须有空格 | (如出现在题名中) | — |

**检查要点**：
- 扫描参考文献中所有英文字母组成的文本片段。
- 检查每个英文标点（`,`, `.`, `:`, `;`）后面是否是空格或行尾。
- 如果标点后紧跟英文字母/数字而无空格，报错。
- 条目的最后一个 `.` 后面可以是行尾/换行，不强制要求空格。

**注意**：中文文本中的标点不适用此规则（中文上下文中标点后不需要空格）。

## 三、文献类型标识

| 标识 | 类型 | 标识 | 类型 |
|------|------|------|------|
| `[M]` | 普通图书 | `[P]` | 专利 |
| `[C]` | 论文集 | `[S]` | 标准 |
| `[G]` | 汇编 | `[N]` | 报纸 |
| `[D]` | 学位论文 | `[A]` | 档案 |
| `[J]` | 期刊 | `[R]` | 报告 |
| `[EB/OL]` | 在线电子资源 | `[M/OL]` | 在线图书 |
| `[R/OL]` | 在线报告 | | |

## 四、各类型参考文献格式细则与样例

### 4.1 普通图书 [M]

**格式**：`责任者. 书名[M]. 出版地: 出版者, 出版年: 页码.`

**正确样例**：
```
李祥浩. 青藏高原东缘环境与生态[M]. 成都: 四川大学出版社, 2002: 20.
```

**检查要点**：
- 有 `[M]` 标识
- 包含：责任者、书名、出版地、出版者、出版年
- 页码可选，如有则格式为 `: 页码`
- 以 `.` 结尾

### 4.2 在线图书 [M/OL]

**格式**：`责任者. 书名[M/OL]. 出版地: 出版者, 出版年[引用日期]. 获取和访问路径.`

**正确样例**：
```
赵耀东. 新时代的工业工程师[M/OL]. 台北: 天下文化出版社, 1998[1998-09-26]. http://www.ie.nthu.edu.tw/info/ie.newie.htm.
```

**检查要点**：
- 有 `[M/OL]` 标识
- 包含引用日期，格式为 `[YYYY-MM-DD]`
- 包含 URL 获取路径

### 4.3 论文集 [C] 和汇编 [G]

**格式**：`责任者. 论文集名[C]. 出版地: 出版者, 出版年.`

**正确样例**：
```
辛希孟. 信息技术与信息服务国际研讨会论文集: A集[C]. 北京: 中国社会科学出版社, 1994.
中国职工教育研究会. 职工教育研究论文集[G]. 北京: 人民教育出版社, 1985.
```

**检查要点**：
- 分别使用 `[C]`（会议录）或 `[G]`（汇编）标识
- 包含出版地、出版者、出版年

### 4.4 报告 [R] / [R/OL]

**格式**：`责任者. 报告名[R]. 出版地: 出版者, 出版年.`

**正确样例**：
```
中华人民共和国国务院新闻办公室. 国防白皮书: 中国武装力量的多样化运用[R/OL]. (2013-04-16)[2014-06-11]. http://www.mod.gov.cnlaffair/2013-04/16lcontent4442839.htm.
World Health Organization. Factors regulating the immune response, report of WHO Scientific Group[R]. Geneva: WHO, 1970.
```

**检查要点**：
- 使用 `[R]` 或 `[R/OL]` 标识
- `[R/OL]` 需要引用日期和 URL
- 在线报告发布日期格式 `(YYYY-MM-DD)`

### 4.5 学位论文 [D]

**格式**：`责任者. 论文题目[D]. 出版地: 出版者（学校/机构）, 出版年.`

**正确样例**：
```
林钰婷. 台产攀蜥属之细胞遗传研究[D]. 台湾: 台湾师范大学生命科学研究所, 2006.
```

**检查要点**：
- 有 `[D]` 标识
- 出版者通常为学位授予单位
- 包含出版地、出版年

### 4.6 专利文献 [P]

**格式**：`责任者. 专利名称: 专利号[P]. 公告日期或申请日期.`

**正确样例**：
```
张凯军. 轨道火车及高速轨道火车紧急安全制动辅助装置: 201220158825.2[P]. 2012-04-05.
```

**检查要点**：
- 有 `[P]` 标识
- 包含专利号
- 日期格式 `YYYY-MM-DD`

### 4.7 标准文献 [S]

**格式**：`责任者. 标准名称: 标准号[S]. 出版地: 出版者, 出版年: 页码.`

**正确样例**：
```
全国信息与文献标准化技术委员会. 文献著录: 第4部分非书资料: GB/T 3792.4-2009[S]. 北京: 中国标准出版社, 2010: 3.
```

**检查要点**：
- 有 `[S]` 标识
- 包含标准号
- 包含出版地、出版者、出版年

### 4.8 专著中析出文献 [M]//...

**格式**：`析出责任者. 析出题名[M]//专著责任者. 专著题名: 其他信息. 出版地: 出版者, 出版年: 页码.`

**正确样例**：
```
宋史卷三: 本纪第三[M]//宋史: 第1册. 北京: 中华书局, 1977: 49.
```

**检查要点**：
- 使用 `[M]//` 连接析出文献和源文献
- 包含源文献信息和页码

### 4.9 报纸 [N]

**格式**：`责任者. 文章名[N]. 报纸名, 出版日期(版次).`

**正确样例**：
```
丁文详. 数字革命与竞争国际化[N]. 中国青年报, 2000-11-20(15).
```

**检查要点**：
- 有 `[N]` 标识
- 日期格式 `YYYY-MM-DD`
- 版次写在日期后的括号内

### 4.10 档案 [A]

**格式**：`责任者. 档案名[A]. 出版地: 出版者, 出版年.`

**正确样例**：
```
中国第一历史档案馆, 辽宁省档案馆. 中国明朝档案总汇[A]. 桂林: 广西师范大学出版社, 2021.
```

**检查要点**：
- 有 `[A]` 标识

### 4.11 期刊 [J]

**格式**：`责任者. 文章题名[J]. 期刊名, 出版年, 卷号(期号): 起止页码.`

**正确样例**：
```
李炳穆. 韩国图书馆法[J]. 图书情报工作, 2008, 52(6): 6-21.
Bravo H, Olavarria J, Torrealba F. Comparative study of visual inter and intrahemispheric cortico-cortical connections in five native Chilean rodents[J]. Anat Embryol(Berl), 1990, 181: 67-73.
```

**检查要点**：
- 有 `[J]` 标识
- 包含：期刊名、出版年、卷号(期号)、起止页码
- 格式：`期刊名, 年, 卷(期): 页码.`
- 英文期刊名及其他英文文本需遵守**英文标点空格规则**

### 4.12 电子资源 [EB/OL]

**格式**：`责任者. 资源名[EB/OL]. (发布日期)[引用日期]. 获取和访问路径.`

**正确样例**：
```
萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15]. http://www.creader.com/news/2001l219/200112190019.html.
Online Computer Library Center, Inc. History of OCLC[EB/OL]. [2000-01-08]. http://www.oclc.org/about/history/default.htm.
```

**检查要点**：
- 有 `[EB/OL]` 标识
- 发布日期格式 `(YYYY-MM-DD)`，可选
- 引用日期格式 `[YYYY-MM-DD]`
- 必须包含 URL

## 五、检查流程

收到参考文献后，按以下步骤逐条检查：

### 步骤 1：识别文献类型
- 根据 `[类型标识]` 确定文献类型。
- 如果没有类型标识，报告错误。

### 步骤 2：检查必填字段
- 根据对应类型的格式细则，检查必填字段是否完整。
- 常见缺失：缺少出版地、缺少出版者、缺少出版年、缺少页码（期刊必需）。

### 步骤 3：检查标点符号
- 中文文献：检查是否一致使用半角标点（`.`, `,`, `:`）。
- 英文文本：检查英文标点后是否有空格（见"英文标点符号空格规则"）。

### 步骤 4：检查日期格式
- 出版日期：`YYYY-MM-DD` 或仅 `YYYY`
- 引用日期：`[YYYY-MM-DD]`
- 发布日期（在线资源）：`(YYYY-MM-DD)`

### 步骤 5：检查 URL
- 在线资源必须有 URL。
- URL 应可访问（提示但可不强制验证）。

### 步骤 6：输出检查结果
- 逐条列出问题，格式：`[行号] 问题描述`
- 若无问题，输出通过标记。
- 最后给出汇总统计：总条数、通过数、问题数。

## 六、常见错误示例

| 错误类型 | 错误示例 | 正确写法 |
|----------|----------|----------|
| 缺少类型标识 | `李祥浩. 青藏高原东缘环境与生态. 成都: 四川大学出版社, 2002: 20.` | `李祥浩. 青藏高原东缘环境与生态[M]. 成都: 四川大学出版社, 2002: 20.` |
| 英文标点缺空格 | `Bravo H,Olavarria J,Torrealba F.` | `Bravo H, Olavarria J, Torrealba F.` |
| 英文标点缺空格 | `Anat Embryol(Berl),1990,181:67-73.` | `Anat Embryol(Berl), 1990, 181: 67-73.` |
| 期刊缺少卷期页码 | `李炳穆. 韩国图书馆法[J]. 图书情报工作, 2008.` | `李炳穆. 韩国图书馆法[J]. 图书情报工作, 2008, 52(6): 6-21.` |
| 中文用全角标点 | `李祥浩．青藏高原东缘环境与生态[M]．成都：四川大学出版社，2002：20．` | `李祥浩. 青藏高原东缘环境与生态[M]. 成都: 四川大学出版社, 2002: 20.` |
| 责任者间分隔不对 | `Bravo H、Olavarria J、Torrealba F.` | `Bravo H, Olavarria J, Torrealba F.` |
| 缺少出版地 | `李祥浩. 青藏高原东缘环境与生态[M]. 四川大学出版社, 2002: 20.` | `李祥浩. 青藏高原东缘环境与生态[M]. 成都: 四川大学出版社, 2002: 20.` |

## 七、特殊注意事项

1. **中英文混排**：一条参考文献中可能既有中文也有英文，英文部分需遵守英文标点空格规则。
2. **三个作者以上**：样例中最多三个作者，超过三个应写为 `作者1, 作者2, 作者3, 等.`（中文）或 `Author1, Author2, Author3, et al.`（英文）。
3. **无明确出版者**：可用 `[出版者不详]` 标注。
4. **无明确出版日期**：可用 `[出版年不详]` 标注。
