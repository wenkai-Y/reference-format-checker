# 含常见错误的参考文献示例

以下条目均有不同的格式问题,可用于测试 skill 是否能正确识别错误。每条后面括号注释了**应该被发现**的问题。

## 错误类型集合

[1] 李祥浩. 青藏高原东缘环境与生态. 成都: 四川大学出版社, 2002: 20.
    (缺少文献类型标识 [M])

[2] Bravo H,Olavarria J,Torrealba F. Comparative study[J]. Anat Embryol, 1990, 181: 67-73.
    (作者间英文逗号后缺空格)

[3] 王某某. 某某研究[J]. 某某学报, 2020,15(3):45-60.
    (英文逗号、冒号后缺空格)

[4] 李炳穆. 韩国图书馆法[J]. 图书情报工作, 2008.
    (期刊缺少卷号、期号、页码)

[5] 李祥浩．青藏高原东缘环境与生态[M]．成都：四川大学出版社，2002：20．
    (中文使用全角标点,应为半角)

[6] Bravo H、Olavarria J、Torrealba F. Some study[J]. Some Journal, 1990, 181: 67-73.
    (英文作者间应该用逗号 + 空格,不应使用顿号)

[7] 李祥浩. 青藏高原东缘环境与生态[M]. 四川大学出版社, 2002: 20.
    (缺少出版地)

[8] 张三. 某资源[EB/OL]. http://www.example.com.
    (在线资源缺少引用日期)

[9] 萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15].
    (在线资源缺少 URL)

[10] 林钰婷. 台产攀蜥属之细胞遗传研究[D]. 2006.
     (学位论文缺少出版地和出版者/学位授予机构)
